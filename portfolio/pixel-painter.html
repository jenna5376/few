Introduction

Pixel Painter is a pixel art program created using Pygames. There are 21 colors and 3 brush sizes, as well as a variety of tools to choose from. There are 6 tools in the program: pencil, eraser, fill tool, color picker, grid, and a clear button. You can also download the finished image as a png file with a custom file name.
